Выход из метро "Няміга" в сторону Дворца спорта (см. карту ниже) является местом встречи для большинства наших экскурсий. Нашего гида можно узнать по оранжевой майке.
Сбор группы для экскурсии по Лошицкому парку - возле центрального входа в парк (со стороны Камвольного комбината, см. карту ниже).
Для некоторых редких маршрутов место встречи оговаривается отдельно в анонсе экскурсии, о нём всегда можно узнать, связавшись с нами.
Для индивидуальных (частных) экскурсий место встречи оговаривается с клиентом.

Возможен прокат и доставка велосипедов. Средняя цена проката 1 велосипеда на 3 часа с доставкой - 10 рублей. Tочная цена зависит от продолжительности маршрута, места встречи, количества нужных вам велосипедов - свяжитесь с нами, чтобы узнать об условиях.
Внимание! В наших экскурсиях можно участвовать на любом технически исправном велосипеде!
